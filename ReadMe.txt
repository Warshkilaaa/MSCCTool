Привет! Данная программа предназначена для тестирования уязвимостей на вашем майнкрафт сервере.
За использование вами программы не по назначению ответствености я не несу. Урок по установке:
1.Установите python
2.Запустить командную строку от имени администратора (надеюсь это умеют все)
3.Напишите в командную строку следующие команды:
python -m pip install –upgrade pip
pip install colorama
